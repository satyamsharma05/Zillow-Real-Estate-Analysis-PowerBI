# 🏡 Zillow Real Estate Market Analysis 2026

![Dashboard Screenshot](Images/dashboard_screenshot.png)

## 📊 Project Overview
Where did this data come from? Not from a ready-made CSV! 
I extracted raw JSON data directly from network requests to build a real-world Real Estate Dashboard. This project analyzes 470+ property listings to uncover market trends, pricing strategies, and "Hot Deals".

**Live Dashboard:** [Link to Power BI Service if published] *(Optional)*

## 🛠️ Tech Stack Used
* **Data Extraction:** Network Traffic Analysis (JSON)
* **Data Cleaning:** Python (Pandas), VS Code, Power Query
* **Visualization:** Microsoft Power BI
* **Analysis:** DAX (Data Analysis Expressions)

## 🔍 Key Features
* **Real-time Data Extraction:** Captured hidden API endpoints.
* **Price Heatmaps:** Used Azure Maps to visualize high-value areas.
* **Dynamic Filtering:** Slicers for City, Beds, Price, and Home Type.
* **Advanced Analytics:** * Decomposition Tree for root cause analysis of price.
    * Key Influencers to identify price drivers.
    * Custom DAX measures for "Avg Days on Market".

## ⚙️ How to Run This Project
1.  Clone this repository.
2.  Open `Zillow Analysis.pbix` in Power BI Desktop.
3.  Refresh the dataset (ensure the JSON file path is correct in Data Source Settings).
4.  Explore the insights!

## 📈 Insights Delivered
* Identified that **Austin** has the highest volume of premium listings (>$1M).
* 4-Bedroom homes have the best Price-to-Rent ratio.
* "Hot Deals" algorithm identifies undervalued properties instantly.
